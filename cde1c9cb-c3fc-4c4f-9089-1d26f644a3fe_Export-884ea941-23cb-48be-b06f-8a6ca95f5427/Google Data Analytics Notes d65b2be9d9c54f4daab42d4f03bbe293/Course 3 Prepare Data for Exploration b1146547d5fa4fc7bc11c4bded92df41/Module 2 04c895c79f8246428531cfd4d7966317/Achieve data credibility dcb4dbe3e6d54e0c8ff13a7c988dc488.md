# Identify good data sources

### **Key Points**

- **ROCCC**: A mnemonic for identifying good data sources:
    - **R**: Reliable - accurate, complete, unbiased, and vetted.
    - **O**: Original - validated with the original source.
    - **C**: Comprehensive - contains all critical information.
    - **C**: Current - relevant and up-to-date.
    - **C**: Cited - credible and verifiable.
- **Good Data Sources**:
    - Public data sets
    - Academic papers
    - Financial data
    - Governmental agency data
- **Importance of Good Data**:
    - Increased confidence in decisions
    - Improved analysis and insights
    - More accurate results

### **Definitions**

- **Data Source**: The origin of data, such as a website, database, or research paper.
- **Reliable**: Data that is accurate, complete, and unbiased.
- **Original**: Data that comes directly from the source, not a third-party.
- **Comprehensive**: Data that contains all the necessary information for analysis.
- **Current**: Data that is up-to-date and relevant to the task at hand.
- **Cited**: Data that has a clear source and can be verified.

# What is “bad” data?

### **Key Points**

- **ROCCC:** A mnemonic for identifying good data sources. Good data should be **Reliable, Original, Comprehensive, Current, and Cited**.
- **Bad data sources:** Data that is not reliable, original, comprehensive, current, or cited. This data can be inaccurate, incomplete, biased, misleading, or outdated.
- **Examples of bad data:**
    - **Unreliable data:** Data with sample selection bias, misleading visualizations.
    - **Not original data:** Data from second or third-party sources without access to the original source.
    - **Not comprehensive data:** Data missing important information or containing human error.
    - **Not current data:** Outdated and irrelevant data.
    - **Not cited data:** Data without a source or citation.

### **Definitions**

- **Reliable:** Data that is accurate, complete, and unbiased.
- **Original:** Data that comes directly from the source.
- **Comprehensive:** Data that includes all the necessary information.
- **Current:** Data that is up-to-date and relevant.
- **Cited:** Data that has a source or citation.

### **Additional Points**

- Bad data can have serious and lasting impacts, leading to incorrect conclusions, inaccurate information, and failed processes.
- Good data sources include vetted public data sets, academic papers, financial data, and governmental agency data.
- The U.S. Census Bureau is a reliable source of regularly updated data.
